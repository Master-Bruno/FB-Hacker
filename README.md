# FB-Hacker
Fb Hacking Tool (free)



```bash
$ apt update
$ apt upgrade
$ pkg install git
$ pkg install python
$ pkg install python2
$ pip2 install mechanize
$ pip2 install requests
$ git clone https://github.com/achuzachu/FB-Hacker
$ cd FB-Hacker
$ ls
$ git pull
$ chmod +x *
$ python Hijack.py

 Tool Installation successful

```
